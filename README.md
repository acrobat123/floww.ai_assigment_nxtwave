API documentation.

POST /transactions : to create a new transaction in the trasactions table
GET /transactions : to  retrieves all transactions in the transactions table.
GET /transactions/:id : to retrieves a transaction by ID in the transactions table.
PUT /transactions/:id : to updates a transaction by ID in the transactions table.
DELETE /transactions/:id : to deletes a transaction by ID in the transactions table.
GET /summary: Retrieves a summary of transactions, such as total income, total expenses, and balance. Optionally, this can be filtered by date range or category.
